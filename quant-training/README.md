# Quant Training Repo

This repo is designed to build deep intuition for:
- Probability
- Expected value
- Decision-making under uncertainty

## Methodology

1. Solve problem analytically
2. Simulate it
3. Compare results
4. Stress test assumptions

## Key Principle

"If you cannot simulate it, you do not understand it."

## Usage

- Solve problems in /problems
- Check /solutions
- Run simulations in /simulations
